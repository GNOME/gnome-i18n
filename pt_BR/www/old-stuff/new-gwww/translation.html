Projeto de Tradução do Gnome

O Projeto de Tradução do Gnome para o português falado no Brasil é parte da LDP-BR e tem como objetivo permitir que o usuário brasileiro possa usar o excelente ambiente Gnome, abrindo as portas para a maior utilização de programas livres no Brasil.

Esta página tem dois objetivos: informar o usuário brasileiro do Gnome sobre a quantas anda o esforço de tradução e convidar os interessados a participar deste esforço. Quem estiver interessado em colaborar com as traduções encontra aqui sugestões de tarefas e referências a documentação útil para quem está querendo começar a traduzir programas livres.

Estado da Tradução

Os programas do Gnome têm desde o princípio um bom suporte para internacionalização. Isto significa que é possível, com relativa facilidade, localizar um programa do Gnome para as convenções sociais e linguísticas de uma sociedade. De forma geral, é possível dizer que esta localização é feita em três partes: a adaptação de convenções locais (moeda, formato da data e hora, formato dos números, etc), tradução da interface dos programas e tradução dos manuais dos programas.

Atualmente, o esforço por parte dos voluntários está sendo concentrado nas duas primeiras tarefas. Isto é feito por meio da tradução de catálogos de mensagens, que são um apanhado de todas as frases visíveis ao usuário de um programa. Quando a tradução do catálogo de mensagens de um programa está completo, seja ele com interface gráfica ou não, toda a interação com o usuário é feito em sua língua nativa, usando as convenções de seu país.

O estado atual da tradução dos catálogos de mensagens dos programas do Gnome pode ser visto aqui, onde o código ISO para o português brasileiro é pt_BR. No momento estamos nos aproximando de 75% de todas as frases traduzidas, o que eleva o português falado no Brasil ao estado de língua oficialmente suportada pelo Gnome.

Uma outra frente essencial para a adaptação completa do Gnome para os usuários brasileiros é a tradução de manuais. Infelizmente, ainda não foi possível começar de modo organizado a resolver este problema. Se você estiver interessado em nos ajudar nesta tarefa, veja mais embaixo como participar do Projeto de Tradução do Gnome.

Onde Conseguir as Traduções

Todas as traduções realizadas pelos membros do projeto são incluídas o mais cedo possível no repositório oficial de códigos fonte dos programas do Gnome e são normalmente disponibilizadas a cada nova versão dos programas. Para conseguir uma versão traduzida do programa você pode:

  • Atualizar as versões de seus programas, as versões mais novas possivelmente já conterão as traduções. Se a sua distribuição não fornece atualizações freqüentes, você pode pegar os fontes das versões oficiais dos programas no ftp do Gnome.
  • Acessar o CVS do Gnome e pegar a versão mais atualizada possível dos programas. Se você tem espírito de aventura, instruções podem ser encontradas ftp do Gnome. Normalmente, estas versões não estão ainda estáveis e seu uso não é recomendado.

O próximo lançamento em grande escala dos programas do Gnome, chamado Gnome 1.4, deverá conter todas as traduções geradas pelos voluntários do projeto até o momento. Esperamos estar muito próximos de 100% para este lançamento.

Quando conseguirmos caminhar mais um pouco na tradução dos manuais, eles poderão ser obtidos junto com os programas traduzidos. Também serão disponibilizadas versões ???on-line??? nesta página.

Como Ativar as Traduções

Como Participar

No momento estamos precisando desesperadamente de voluntários para ajudar na tradução dos programas e documentos. Se você está disposto a ajudar, dê uma olhada nas tarefas prioritárias e não hesite em entrar em contato.

Para ajudar, basta ter conhecimento da língua inglesa, alguma familiaridade com os programas sendo traduzidos e muita vontade de colaborar para a disseminação de programas livres entre os usuários brasileiros. Mande uma mensagem para a lista ldp-br informando que tipo de tradução você está interessado em fazer, o responsável pela coordenação das traduções irá te passar algumas sugestões de tarefas de tradução e você já pode começar a traduzir.

Contato

Existem várias maneiras de entrar em contato com os participantes do Projeto de Tradução do Gnome:

  • Lista ldp-br: A lista de discussão por correio eletrônico da LDP-BR é o principal meio de comunicação entre os participantes do projeto. Para assiná-la, clique aqui.
  • Canal de IRC: Vários participantes do projeto de tradução podem ser encontrados no canal de IRC oficial do LDP-BR. Venha bater um papo conosco no irc.linux.org, canal #ldp-br. Você pode usar a interface web para este canal aqui.

Tarefas

Estas são algumas das tarefas que estão necessitando de atenção no momento. É interessante que a maioria delas esteja completa até o fim do ano, para quando está prevista a primeira versão do Gnome 2.0. Se você estiver interessado em ajudar, entre em contato com o coordenador do projeto.

  • Tradução do Gnome Users Guide: O principal manual do usuário do Gnome é a porta de entrada para novos usuários. Um documento importante e longo que exige grande dedicação do tradutor.
  • Tradução do Gnumeric: O Gnumeric é a planilha de cálculos do Gnome Office. Este programa é um sério competidor do Microso~1 Excel e sua tradução é muito importante para vários usuários. Este projeto exige um conhecimento de planilhas de cálculos e disposição para traduzir mais de 1000 frases.
  • Tradução dos programas: O projeto Gnome já dispõe de vários programas a disposição dos usuários, falta agora traduzi-los para os usuários brasileiros. As traduções têm de 100 a 1000 frases e o esforço necessário é ajustável ao seu tempo livre, sendo requisito apenas o conhecimento do programa em questão.
  • Tradução dos manuais: A maioria dos programas do Gnome possui um manual, o que é muito importante para os usuários novatos. Traduzi-los é um projeto de longo prazo a espera de voluntários.
