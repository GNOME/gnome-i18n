Projeto Gnome Brasil

O Projeto Gnome visa construir uma interface gráfica e um ambiente de desenvolvimentos de programas para Unixes baseada no uso de programas livres e novas tecnologias. Esta é a página do Projeto Gnome Brasil, aqui poderão ser encontradas algumas informações sobre o projeto em português, para informações mais completas em inglês visite a página oficial do projeto.

O esforço para adaptar o Gnome para a língua portuguesa faz parte de um projeto maior para tornar o Linux mais amigável para usuários brasileiros o LDP-BR (Linux Documentation Project - Brasil).

Projeto de Tradução do Gnome

No momento, a principal atividade dos gnomes brasileiros é a tradução dos programas e manuais do Projeto Gnome. Se você está disposto a nos ajudar ou apenas está curioso para saber como andam as traduções visite a página do projeto de tradução.
