# Esta é uma lista de dicas para o SCREEM. Toda vez que o SCREEM for iniciado,
# uma dica será selecionada e mostrada na caixa "Dica do dia".
# (Da mesma forma que no GIMP)
#
# O formato de arquivo é idêntico aos arquivos de dica do GIMP
#
# - Linhas que iniciam com '#' são comentários.
# - Linhas em branco ou comentários separam duas dicas
#   (estas não são ignoradas).
#   Múltiplas linhas em branco são tratadas como uma. Se você deseja ter uma 
#   linha em branco em uma dica, coloque um espaço ou aba na mesma.
# - O texto aparecerá na caixa como está neste arquivo.  Isto é feito de 
#   propósito para obter-se mais liberdade na disposição das dicas do que
#   com o envolvimento automático de palavra, mas isto significa também que
#   linhas excessivamente longas devem ser evitadas neste arquivo.
# - As dicas devem ser concisas: 3 linhas ou menos.
#
#--------------------------------------------------------------------------

# A primeira dica deve ser uma mensagem de boas vindas, porque esta é a 
# primeira coisa coisa que um novo usuário verá.

Bem-vindo ao SCREEM !
 
Favor ler as notas de divulgação que são mostradas na janela de pré-visualização

Podem ser configuradas as cores a serem usadas para destacar a sintaxe
acessando as Abas Coloridas na aba Configurações->Preferências

Com a marcação em linha (inline) ativada, ao pressionar espaço no interior
de uma tag surgirá um menu com todos os atributos válidos a serem escolhidos.
Se você não deseja o menu, apenas continue digitando e ele desaparecerá.

Marcas feitas no editor são inseridas na altura que elas são na verdade
marcadas em vez da posição atual do cursor.

Arrastando um arquivo da árvore de arquivos para o editor, irá inserir o caminho
daquele arquivo relativo na página atual.

Arrastando com o botão central do mouse da árvore de arquivos para o editor,
um menu de opções sobre o que fazer será apresentado.

Um clique direito em uma tag adiciona um ítem de menu ao menu que permite
adicionar / alterar os atributos de tag.

Algumas tags, quando recebem um clique direito adicionam uma opção ao menu, 
desta forma pode-se usar o "Wizard" para aquela tag.

Inserção de auto-entidade irá inserir entidades html quando apropriado, ex.
&aspas; serão inseridas em vez de uma ". A entidade & é realizada duas vezes.

Quando movendo arquivos via árvore de arquivo quaisquer links em suas páginas
serão automaticamente atualizados para apontar para a nova localidade.

Aplicações de ajuda, quando executadas, irão tomar a página atual no editor,
e mostrar o resultado de processamento de volta ao editor, atuando como um
filtro.

Pode-se adicionar/visualizar/editar tarefas que necessitam desempenho em seu 
site, usando o botão da lista de pendências na barra de tarefas.

O editor modelo pode ser usado para criar modelos, o qual cria uma
completa hierarquia de site quando usado na criação destes, úteis se desejamos
sites com um visual/disposição semelhantes.

Pode-se rapidamente acionar páginas usando entre maiúsculas e minúsculas para
as tags usando as opções no menu de edição.
